

缘起~~
很多的时候我们DIY三极管电路的时都只能依靠厂方的数据手册作为依据来设计，可是我们买到的三极管却是千差万别各不相同，这如同盲人骑瞎马，电路性能完全只能凭运气。作为一个电子爱好者我连一个能测三
极管放大倍数的万用表都没有，怎么说呢，一个字苦恼~~~~无助，空虚寂寞冷~~~
市面上有一些晶体管图示仪，一般都是全模拟的技术的 使用阴极射线管x,y模式的晶体管显示晶体管图示仪，只能测三极管，对JFET无能为能力，而且笨重，电流大小靠眼看，靠心估，如今时代洪流早已跨入信息化的时代，我们
是不是需要一台便宜好用的容易自己制作的晶体管特性测试仪呢？恰好 我自己懂一些编程，何不自己开干DIY一台呢？这个想法是今年5月开始的，接连打了5块板子，其间也各种测试实验，折腾，如今总算成了，，
，现把电路图和单片机源代码，，以方便大家制作，并附上上位机程序方便大家直接使用。

简介

MY晶体管特性测试仪。使用STM32F072C8T6作为主控，使用12Mhz USB连接电脑,PC机程序作为显示终端并作为人机操作接口
主要特性：
	1,  全固态开关，没有任何机械继电器,扫描速度快典型单条曲线扫描60ms即可完成。
	2,  全自动档位选择，自动选择采样电阻，放大器增益组合。
	3，使用TDA2030作为可编程电压输出（+-9），输出电流可达2A以上  
	4，4档采样电阻。3档放大器增益。可以使曲线Uce更接近目标电压。
	5，如果你没有声卡的时候还可以把它当成一个声卡，还能直接直接驱动音箱~~~无敌了~~~哈哈~~~~
NOTE
1变压器中间抽头最好接地以获得更好信噪比。
2.当使用测试大功率管时，你应该确保你的供电有足够大输出电流（如果有大电流需求，而变压器不能提供时，可以使用加二极管并联
直流电源的方式提供额外的电流供应）
3.TDA2030散热器壳与VEE相联而不是GND，不要与GND发生短路。
4.JFET-P没有做过测试，不能保证能正常使用，本是买了些MMBFJ177来做测试的，哪知道遇到黑心商家，1，2脚有个二极管这哪是JFET嘛
